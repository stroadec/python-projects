# Python Task Automator

## Description
A Python script that automates the cleaning and processing of CSV and Excel files.

This tool is designed for businesses or freelancers who receive repetitive, messy data files and want clean, structured outputs automatically.

## Features
- Load CSV or Excel files
- Clean and normalize data
- Handle missing values
- Export clean files automatically

## Example use case
A company receives weekly Excel reports with inconsistent formatting.
This script processes the file and outputs a clean, ready-to-use version.

## How to run
```bash
pip install -r requirements.txt
python main.py
