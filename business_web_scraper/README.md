# Business Web Scraper

## Description
A Python web scraper that collects public data from websites and exports it
into structured CSV files.

This project demonstrates how to extract, clean, and export web data
for business use cases such as monitoring, analysis, or reporting.

## Features
- HTTP requests with error handling
- HTML parsing with BeautifulSoup
- Structured data extraction
- CSV export

## Legal Note
This scraper is demonstrated on a public website designed for scraping.
Always respect website terms of service.

## How to Run
```bash
pip install -r requirements.txt
python main.py
